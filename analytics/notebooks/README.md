# Notebooks

Denne mappe er reserveret til notebooks i projektet.

Den aktive predictive maintenance notebook ligger i:

```text
analytics/ml/predictive_maintenance.ipynb
```

---

## Formål

Notebook-delen demonstrerer, hvordan operationsdata kan bruges til machine learning i VoltEdge MVP’en.

Machine learning-delen fokuserer på predictive maintenance, hvor målet er at identificere ladestandere med forhøjet risiko for vedligeholdelse.

---

## Afgrænsning

Data er syntetiske og bruges kun til MVP-formål.

Notebooken skal derfor ikke forstås som en produktionsklar ML-model, men som et proof-of-concept, der viser hvordan VoltEdge kan arbejde mere datadrevet med vedligeholdelse og drift.
